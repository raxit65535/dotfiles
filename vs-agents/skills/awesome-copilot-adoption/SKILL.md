# Awesome Copilot Adoption

Use this skill to import external agent/skill content in controlled phases.

## Principles

- Import one capability area at a time.
- Prefer adaptation over copy-paste.
- Validate behavior with one realistic prompt.
- Remove or tune assets that add routing noise.

## Suggested Adoption Sequence

1. Planning and review capabilities
2. Implementation-oriented capabilities
3. Validation and test capabilities
4. Domain-specific specialists

## Per-Import Checklist

1. Record source and intended outcome.
2. Normalize naming and frontmatter for your workspace.
3. Add one smoke test scenario.
4. Review results after real usage and refine.
