# OpenSpec Templates

This folder provides reusable templates for planning non-trivial changes.

## Included Templates

- `feature-spec-template.md`
- `rollout-checklist-template.md`
- `risk-register-template.md`

## How to Use

1. Copy `feature-spec-template.md` into the target repo's OpenSpec change.
2. Fill scope, alternatives, and validation details.
3. Use `risk-register-template.md` for high-risk changes.
4. Use `rollout-checklist-template.md` before deployment.

OpenSpec source of truth remains in the target project repository.
