# Rollout Checklist

## Pre-Deploy

- [ ] Scope confirmed with stakeholders
- [ ] Validation suite green
- [ ] Backward compatibility reviewed
- [ ] Monitoring and alerts prepared

## Deploy

- [ ] Deploy in controlled environment first
- [ ] Validate key paths manually
- [ ] Monitor error rate and latency
- [ ] Confirm dependent services are healthy

## Post-Deploy

- [ ] Verify success metrics
- [ ] Confirm no regression signals
- [ ] Communicate status

## Rollback

- [ ] Rollback command/procedure documented
- [ ] Rollback tested or previously validated
- [ ] Trigger conditions defined
